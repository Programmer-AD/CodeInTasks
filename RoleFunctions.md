# Functions by roles
## Guest (unauthorized)
- View and filter tasks
- Register
- Login

## User (authorized)
- Add task solutions
- View solution check status

## Creator
- Create task
- Edit own task
- Delete own task

## Manager
- Set/unset Creator role
- Edit any task
- Delete any task
- Ban/unban user

## Admin
- Set/unset Manager/Admin role

## Builder (special, non-people):
- Publish solution status
